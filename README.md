# Gmail Login Test Case (Manual Testing)

## Project Description
This project contains manual test cases for verifying the Gmail login functionality.

The objective of this test case is to verify whether the system successfully navigates the user to the Gmail inbox after entering valid login credentials.

## Test Scenario
Verify Gmail login functionality with valid email and password.

## Test Steps
1. Open the browser.
2. Enter the URL: www.gmail.com
3. Enter a valid email address.
4. Click on the Next button.
5. Enter the valid password.
6. Click on the Next button.
